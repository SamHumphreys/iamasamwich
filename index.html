Cookr

Tools and tech used:

Description:

Cookr is an app designed to help users organise dinner parties by hiring chefs who have registered with the site. A chef can upload a list of menu options that they have available, and the dinner hosts can search through the menus that have been uploaded and create an event where the relevant chef will cook the selected menu for the host.

This app was created as a group project, and my main contribution was the messaging service that the app contains. When a host creates an event a message is sent to the selected chef, who can then respond or discuss details of the event.

MessageMe/Howlr

Tools and tech used:

Description:

After completing Cookr I created MessageMe/Howlr as a messaging app using Websockets to transfer data to and from the server rather than Ajax polling.

Once a user has logged in they can see a list of all users, view their conversation history with each user and send a new message to another user. The page updates with the number of unread messages, and if a conversation has unread messages the person who sent the message will be coloured red.
